# Telegram Bot

هذا هو بوت تيليجرام يعتمد مكتبة Aiogram للعمل.

## المتطلبات

- Python 3.10+
- مكتبات موجودة في requirements.txt

## الإعداد

1. ضع توكن البوت في Environment Variables
2. ضع ADMIN_ID و CHANNEL في Environment

## التشغيل

```bash
pip install -r requirements.txt
python bot.py
---
